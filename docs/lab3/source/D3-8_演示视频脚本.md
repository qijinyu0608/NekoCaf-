第一步：进入项目目录



### 3.2 片段二：本地测试与服务状态

时间建议：`0:40 - 1:30`

讲解词建议：

“这里先通过单元测试和容器状态确认当前 PoC 是稳定可运行的。”

第一步：运行测试

```bash
./.venv/bin/pytest -q
```

第二步：启动服务

```bash
docker compose up --build -d
```

第三步：检查容器状态

```bash
docker compose ps
```

操作要点：

1. `pytest` 结果应为通过
2. `docker compose ps` 中应看到 `member`、`reservation`、`prometheus`
3. `member` 与 `reservation` 应为 `healthy`
